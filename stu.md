## 学生信息表设计
| 中文名称 | 表名 | 字段属性 | 默认值 | 备注 |
|---------|-----|---------|-------|-----|
| 学号 | id | int(4) | | 为主键不可为空 |
| 姓名 | sname | char(10) | | 不为空 |
| 性别 | sex | char(2) |   | 不为空 |
| 年龄 | age | SMALLINT | 1 | 不为空 |
| 所在院校编码 | sid | int(4) | | 不为空 |
